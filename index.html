I have come up with a new way of keeping LLNL's changes up to date with master. We are currently tracking master with 2 branches

  • master-llnl-upstream
  • master-llnl-only.

NOTE:

These branches are 'rebased' off of master. As such their histories will not be guaranteed for those who may clone them. They are provided as a reference, and to maintain the current patches to the master branch LLNL requires.

master-llnl-upstream

is based off of the master branch and contains changes which we are actively pushing upstream.

master-llnl-only

is based off of master-llnl-upstream and contains additional patches which we need here at LLNL and we may or may not pursue upstream acceptance.

You can also clone the project with Git by running: 

$ git clone git://github.com/weiny2/opensm-llnl

get the source code on GitHub : weiny2/opensm-llnl
